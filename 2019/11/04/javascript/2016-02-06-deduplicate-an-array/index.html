Objects

当元素为对象(Object)时，我们就不能用这种办法了，
因为对象存储的是引用而原始变量存储的是值。

因此我们需要改变一下我们的实现方法，使用哈希表。

因为哈希表在Javascript里是一个简单的Object，它的key永远是String类型。这意味着我们不能区分字符串和数字表示的相同的值，如1和'1'。

然而，因为我们使用的JSON.stringify，String类型的key
将会被存储为一个字符串值，这样hashTable的key就唯一了。

这意味着相同的值，但不同类型的元素，将以原来的格式保留。
