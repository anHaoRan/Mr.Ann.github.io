下文是JavaScript中声明变量的不同方法。
注释与console.log足够说明这里发生了什么：

首先，我们只设置了两个变量。并没有很多。

正如你所看到的，代码只改变了全局的y，因为我们在闭包里并没有声明此变量。

现在我们用var声明了两个变量。意味着他们仅在闭包内有作用。

两个变量都使用var声明了而且在之后又给它们赋值。由于local > global，闭包内声明了x和y，意味着闭包内是无法访问全局的x和y的。
