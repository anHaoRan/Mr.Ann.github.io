Nov 04, 2019zh_CNjavascript点击

文档里说reduce()方法接收一个函数作为累加器（accumulator），数组中的每个值（从左到右）开始合并，最终为一个值。

reduce()

reduce() 函数接收2个参数(M: 必填, O: 可选)：

  • (M) 回调reducer 函数 处理先前的结算结果和下一个元素直到序列结束。
  • (O) 初值 作为第一次调用回调时的第一个参数。

所以，让我们先看一个普通用法，之后再看一个复杂用法。
