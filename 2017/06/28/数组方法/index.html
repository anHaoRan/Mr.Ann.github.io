时间格式

Date参数

new Date

不接收任何参数：返回当前时间；

new Date(x)

接收一个参数x: 返回1970年1月1日 + x毫秒的值。

new Date().getFullYear()

获取当前年份；

new Date().getMonth()

获取当前月份,从0开始

new Date().getDate()

获取当前日期

new Date().getDay()

返回当前是星期几

new Date().getTime()

获取从1970.1.1开始的毫秒数（时间戳）

new Date().getHours()

获取当前小时数（24小时制）

new Date().getMinutes()

获取当前分钟数

new Date().getSeconds()

获取当前秒数

new Date().getMilliseconds()

获取当前毫秒数

new Date().toLocaleDateString()

获取当前日期年月日—–格式0000-0-00

new Date().toLocaleTimeString()

获取当前时间—00：00：00

new Date().toLocaleString()

获取当前日期年月日时分秒—–格式0000-0-00 00：00：00
